\xxx{Implementation note: The prototype is implemented in C++ and designed to be portable across Windows and Linux by keeping most logic in platform-neutral modules and depending on cross-platform libraries. In practice, platform-specific code is concentrated in hardware-facing components (notably camera acquisition) and in the chosen UI/web stack; the remaining sensing, wrappers, use-case logic, and the core runtime are OS-agnostic. we have actually been developing on both windows and linux interchangably, but macOS was not tested.}
