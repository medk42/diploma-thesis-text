\chapter*{Declaration on the Use of Generative Artificial Intelligence}

I declare that the following generative AI tools were used during the preparation of this thesis in the ways stated below:

\begin{itemize}
  \item \textbf{ChatGPT (OpenAI)} was used in the period \textbf{August 2024--April 2026}.
  It was used for brainstorming and refining the thesis scope and chapter structure, for drafting and revising selected passages, for improving wording and clarity, and for consultation on implementation options (e.g., selecting suitable libraries, solvers, and hardware configurations). All generated text and suggestions were reviewed and edited by the author.

  \item \textbf{GitHub Copilot (GitHub/Microsoft)} was used in the period \textbf{August 2025--April 2026}.
  It was used primarily as code completion for repetitive implementation patterns (e.g., serialization/deserialization and similar boilerplate across modules). The final code was authored, reviewed, and validated by the author.

  \item \textbf{Cursor (AI-assisted IDE)} was used in the period \textbf{December 2025--April 2026}.
  It was used to accelerate refactoring and repetitive edits in later development stages (e.g., propagating interface changes across similar modules and updating parameter loading/saving logic). The author reviewed, corrected, and integrated all resulting changes.
\end{itemize}

After using the tools listed above, I thoroughly reviewed and modified the outputs as needed and I fully accept responsibility for the final content of the thesis, including all technical claims, experimental results, and interpretations.
